# 小说专用规则

> 基于 qu-ai-wei 通用去AI味引擎，针对小说场景的专项规则。

---

## 一、对话去书面化

### 问题
AI 生成的对话往往书面、正式、缺乏个性，像新闻稿而非角色台词。

### 规则

1. **加语气词**：根据角色性格添加"吧、呢、啊、嘛、嘿、切"等
2. **按角色区分**：
   - 粗犷角色：短句、脏话、命令式（"放屁""少废话"）
   - 文雅角色：长句、委婉、留余地（"或许可以试试"）
   - 孩童角色：叠词、疑问多、逻辑跳跃
3. **不用对话标签**：不写"他说道""她回答道""他叹了口气说"
4. **对话独立成段**：每句对话单独一段
5. **允许不完整句**：断句、省略、口吃都是正常的

### 对比示例

```
❌ 书面化：
"我认为我们应该尽快离开这里，因为危险正在逼近。"他严肃地说道。

✅ 口语化：
"走。"
他拽住我的胳膊，手劲大得吓人。
"现在就走，别回头。"
```

## 二、结尾去升华

### 问题
AI 倾向于在章节末尾加哲理总结、人生感悟、主题升华。

### 规则

**禁止的结尾方式**：
- ❌ 哲理总结：「他终于明白了，真正的强大不是力量，而是内心的坚定。」
- ❌ 主题升华：「这一刻，他不再是一个渔村少年，而是一个肩负使命的人。」
- ❌ 情感抒发：「风中似乎带着一丝温暖，像是命运的回应。」

**允许的结尾方式**：
- ✅ 动作：「姬无尘加快脚步。沙子还是软的，但他的步子比刚才大了。」
- ✅ 对话：「"走吧。"他对自己说。」
- ✅ 悬念：「胸口的五彩石又热了一下。」
- ✅ 感官：「海风灌进领口，凉飕飕的。」
- ✅ 场景：「白发道人转身，往北边走去。」

## 三、禁词库

### 最毒模式（6星）
- "不是A，而是B" → 直接写B
- 例：「不是害怕，而是愤怒」→「他攥紧了拳头」

### 一级禁词（立即替换）
| 类别 | 禁词 |
|------|------|
| 模态 | 仿佛、犹如、宛若、如同、一丝、一抹 |
| 动作 | 深吸一口气、缓缓、不禁、微微、轻轻、淡淡 |
| 表情 | 眼中闪过、嘴角勾起、眉头微皱、瞳孔微缩 |
| 心理 | 心中一动、心头一震、心下了然、心中暗道 |
| 判断 | 不容置疑、显而易见、毫无疑问、不可否认 |
| 描写 | 坚定、闪烁着光芒、狡黠、深邃、凛冽 |

### 二级禁词（频繁时替换）
- 突然、好像、瞬间
- "他终于明白……"
- 连续3个以上排比段

### 破折号
尽量不用破折号（——），用逗号或句号代替。

---

## 四、比喻规则

1. **每段最多1个比喻**：避免比喻堆叠
2. **不用"如X般"句式**：这是翻译腔，改为"像……"或直接描写
3. **比喻要具体**：来自角色生活环境（渔民用水/鱼/船，猎人用弓/箭/兽）
4. **不用常见比喻模板**：如"像一根刺扎在心里""如潮水般涌来"

---

## 五、叙事节奏

1. **长短交替**：连续3个长句后必须接短句
2. **动作密集时用短句**：打斗、追赶、紧急情况
3. **回忆/描写时可用长句**：但不超过60字
4. **单句成段**：用于强调、停顿、转折
