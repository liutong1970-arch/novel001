# 小说自然度质检流程（L6 质量治理层，wenxin-v6）

本文件用于章节写完后的本地复查和修订。它不追求绕过某个平台，不依赖外部站点；只做作品内部质检和可审计修订。

## 借鉴的成熟思路

- `Humanizer-zh` / `blader/humanizer`：先识别常见写作痕迹，再保留含义做局部改写。
  参考：https://github.com/op7418/Humanizer-zh
- `remove-ai-flavor-writing-sskill`：中文文本要重点清理二分对照壳、机械顺序壳、本质拔高壳、讲义腔、段落同构和假互动。
  参考：https://github.com/B1lli/remove-ai-flavor-writing-skill
- `patina`：检测和改写要能跨语言运行，规则要可解释、可复查，不把文本交给黑盒判断。
  参考：https://github.com/topics/humanizer
- `StealthHumanizer`：有效改写通常是多轮、小范围、风格感知，而不是一次性同义词替换。
  参考：https://github.com/rudra496/StealthHumanizer
- 中文写作 agent 类项目：要注入真实观点、细节和读者反应，但小说里应转化为角色偏心、误判、物件、动作后果和代价。
  参考：https://github.com/topics/ai-writing

吸收方法论，不照搬外部项目文本。小说的自然感来自故事选择，不来自把句子改得更口语。

## 高风险来源

1. 细纲抽象：只有"展示、体现、铺垫、震惊"，没有目标、阻力、行动和代价。
2. 段落太匀：连续短句、连续中等段、每段都是"动作 + 感受 + 总结"。
3. 语言模板壳：不是 A 而是 B、真正重要的是、显而易见、他不禁、微微、缓缓、仿佛。
4. 信息说明密：设定词、机构词、规则词堆在一起，角色没有现场判断。
5. 情感空：角色在"意识到/明白/震惊"，但身体、误判、私人记忆和实际损失缺席。
6. **POV 漂移：写了角色不知道的事（wenxin-v6 新增）**。
7. **重复段落：章内或章间重复率 >80%（wenxin-v6 新增）**。
8. **信息密度低：每 500 字没有信息增量（wenxin-v6 新增）**。
9. **章末钩子弱：结尾不能让人想点下一章（wenxin-v6 新增）**。

## 固定流程

1. 运行本地扫描，看整章节奏、语言、纹理、统计四项。
2. 生成分段队列，只处理高风险片段。
3. 回到细纲判断诱因：若场景没有目标和阻力，先修细纲；若细纲成立，再修正文。
4. 对每个片段做小步改写：打散段落呼吸、换成可见行动、补角色偏心、删讲义腔。
5. 复扫。分数只作为排序工具，最终以读者是否能看见人物在选择为准。
6. **运行 wenxin-v6 检测：重复段落、POV 一致性、章末钩子、信息密度。**
7. **如果追读力 E < 6，无论自然度分数多低，必须重写。**
8. **如果低质风险为 P0/P1，必须先回细纲或结构，不做表层润色。**

## 改写原则

- 保留剧情事实、人物关系、伏笔和视角。
- 不用错别字、乱码、不可见字符、随机口语词制造"人味"。
- 不做同义词替换器；换词不能解决段落同构。
- 不把所有句子切短；需要短句，也需要一两处能承载判断和环境压力的长段。
- 不把小说改成聊天记录；角色声音要随身份、欲望和压力变化。
- **不重复使用同一桥段（如"可乐不凉了"在同一章内不超过 2 次）。**
- **不写 POV 角色不知道的事。**

## 修订动作表

| 风险 | 优先修法 |
| --- | --- |
| 节奏高 | 调整段落长短，插入行动中断或观察延迟 |
| 语言高 | 删除模板壳、高概率词链、总结句和抽象拔高 |
| 纹理高 | 增加具体物件、身体反应、误判、私人记忆或小损失 |
| 统计高 | 降低设定词密度，把规则放进动作结果里 |
| **重复段落** | **删除或重写重复段落（wenxin-v6 新增）** |
| **POV 漂移** | **只写 POV 角色能看到/听到的事（wenxin-v6 新增）** |
| **章末钩子弱** | **改为强悬念/反常事件/情绪高点（wenxin-v6 新增）** |
| **信息密度低** | **每 500 字增加一个信息增量（wenxin-v6 新增）** |
| **低质风险高** | **回到 `quality-governance.md`，先修因果、主动性和代价** |

## 小说专用判断

一段文字更像人写，通常不是因为它"更乱"，而是因为它带着选择痕迹：角色看错了什么，舍不得什么，临时改了什么，付出了什么小代价。修订时优先把这些痕迹写出来。

## 追读力最低标准（wenxin-v6 新增）

每章完稿后，必须满足以下所有条件：

| 检查项 | 最低标准 |
|--------|---------|
| 信息增量密度 | ≥ 0.8/500字 |
| 爽点 | ≥ 1 个/章 |
| 章末钩子强度 | ≥ C 级 |
| 主角主动性 | ≥ 1 次主动行动/章 |
| 重复段落 | 0 处 |
| POV 漂移 | 0 处 |
| 追读力 E | ≥ 6 |
| 低质风险 | 不出现 P0/P1 |

任一不达标，本章不合格。
