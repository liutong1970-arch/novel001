# 番茄爆款风格指纹（L2，新增）

## 声明

本风格为多风格库中的可选项之一。写作时单选一种风格，不混搭。选用本风格时，全文按本指纹生成。

本文件所有范例均按本项目创作律（L1）规范化，非逐字照搬。

## 五维指纹

| 维度 | 偏向 | 说明 |
|------|------|------|
| 突发性 | 高 | 短句为骨架（3-8 字短句高频），关键信息用长句强调，形成强方差 |
| 困惑度 | 中高 | 设定专业词硬性并置，但信息释放有节奏——不一次性倾倒 |
| 信息熵 | 高 | 角色声音高度区分，句式多变，环境描写来自角色生活环境 |
| 追读力 | 极高 | 每 500 字一个信息增量，每章 ≥1 个爽点，章末强钩子 |
| 平台留存力 | 极高 | 书名/简介/第一章强绑定，黄金三章快速兑现类型快感，前 10 章建立追读梯度 |

## 文风内核

### 1. 紧贴主角视角（第一人称或有限第三人称）

- 读者通过主角的眼睛看世界。
- 只写主角能看到、听到、想到的内容。
- 不写配角内心活动，除非通过外部表现推断。
- 配角 POV 段落仅限主角不在场的信息补充，每章不超过 2 段。

### 2. 情绪浓烈直白

- 愤怒、屈辱、狂喜、不甘——这些情绪可以直接表达。
- 与"白描律"不冲突：白描适用于日常场景，冲突/高潮场景允许情绪放大。
- 情绪表达要有出处——不是因为"作者想煽情"，而是因为角色真的经历了什么。

### 3. 节奏快，信息密度高

- 300-500 字一个信息点（新设定/新冲突/新反转/新关系）。
- 减少路人恐惧反应的篇幅（从每章 3-4 次降为 1-2 次）。
- 减少重复的环境描写（雨、青砖、路灯——每章最多 1 次）。
- 主角必须有主动性——不能只被动应对事件。

### 4. 爽点密集

- 每章至少 1 个爽点时刻（打脸/升级/反杀/被认可/捡漏/扮猪吃虎）。
- 每 3-5 章至少 1 个大爽点（身份揭示/实力突破/重大逆转）。
- 压抑-爆发比例：压抑不超过爆发的 1/3。
- 期待感管理：提前埋期待 → 延迟满足 → 充分兑现。

### 5. 章末强钩子

- 每章结尾必须让读者想点下一章。
- 强钩子类型（按强度排序）：新威胁 > 反常事件 > 悬念 > 关系变化 > 对话钩子。
- 禁止的结尾：环境描写、重复性动作、主角发呆、配角恐惧反应。

## 仿写参数

- **视角**：紧贴主角（第一人称或有限第三人称），POV 一致性。
- **主角 POV 占比**：每章至少 70%（引用 L1 紧贴主角律）。
- **情绪**：浓烈直白（愤怒、屈辱、狂喜直接表达），冲突场景允许情绪放大。
- **节奏**：快（300-500 字一个信息点），每章 2-3 个深场景。
- **场景**：2-3 个深场景，每个场景有完整的情绪周期。
- **对白**：带性格标签（口头禅、语气词、说话方式有辨识度），每个角色声音可区分。
- **爽点**：每章 ≥1 个（打脸、升级、被认可、反杀、捡漏、扮猪吃虎）。
- **章末**：强悬念/强情绪钩子。
- **字数**：2000-3000 字。
- **主角主动性**：每章至少 1 次主动选择和行动。

## 范例（规范化后）

### 打脸场景

```text
"就凭你？"赵天霸把玩着手中的玉扳指，嘴角挂着笑。
"就凭我。"路天明把账本合上。
赵天霸笑了。他身后三个保镖也笑了。
路天明没笑。他指了指门槛。
"你踩到了。"
赵天霸低头。鞋尖确实压过了那道浅浅的水线。
"所以？"
"三千。"
"什么？"
"门槛赔偿。三千块。"
赵天霸的笑声停了。
"你说什么？"
路天明翻开账本。笔尖压在纸面上，发出沙沙声。
"踩坏门槛，三千。威胁店主，五千。扰乱营业，一万。"
他抬头，看着赵天霸。
"一共一万八。现金还是转账？"
```

### 章末钩子

```text
路天明把账本合上，放回抽屉里。
他拿起可乐喝了一口，皱眉，放下。
门外，刘德财的车灯在雨里亮着。
但路天明没看车灯。
他看着车灯旁边的那个人——灰色中山装，拐杖，半白头发。
梁秋生。
S级精神系，华夏异能界三大支柱之一。
路天明放下可乐。
"又来个不懂规矩的。"
柜台下的黑犬抬起头。
它的眼睛在暗处闪了一下。
那簇被装在打火机壳里的黑炎，不安地抖了抖。
路天明叹了口气。
"今天已经是第三个了。"
```

### 主角主动行动

```text
路天明没有等沈青霞开口。
他已经把搪瓷杯放在了柜台上。
"一万。"
沈青霞愣了一秒。她以为路天明会先问她要什么。
但路天明不问。他直接把价格写在柜台上，像菜市场卖菜。
"治伤的水，非卖品。"
"那这杯？"
"这杯十万。"
沈青霞还没来得及回答，江勇已经怒了。
路天明看了江勇一眼。
"嫌贵可以不喝。"
```

## 与防 AI 痕迹的天然合流

番茄爆款风格在三维度上的优势：

- 紧贴主角视角 → POV 一致性 → 减少视角漂移（AI 的典型标志）。
- 情绪浓烈直白 → 角色声音区分度高 → 信息熵高。
- 快节奏 + 爽点密集 → 信息增量密度高 → 追读力高。
- 章末强钩子 → 悬念缺口 → 困惑度高。

选番茄爆款风格，"写得好"、"测不出"、"让人追读"三者自动统一。

## 与原风格的区别

| 维度 | 方想风格 | 天魔风格 | 番茄爆款风格 |
|------|---------|---------|-------------|
| 视角 | 第三人称全知 | 第三人称全知 | 紧贴主角（有限第三人称） |
| 情绪 | 极度冷淡 | 中等 | 浓烈直白 |
| 节奏 | 慢（长段铺陈） | 中等 | 快（300-500字信息点） |
| 场景 | 1-3 场景 | 1-3 场景 | 2-3 深场景 |
| 爽点 | 不明显 | 中等 | 每章 ≥1 个 |
| 章末钩子 | 弱 | 中等 | 强 |
| 主角主动性 | 被动 | 被动 | 主动 |
| 对白 | 无标签 | 无标签 | 有性格标签 |
| 适用类型 | 科幻/机甲 | 玄幻 | 都市异能/系统/爽文 |
