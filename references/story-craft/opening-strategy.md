# 开书策略与黄金三章（L3）

本文件用于 `/wenxin-v6 开书体检`、新书立项、前三章审稿和前 10 章规划。开书的目标不是慢慢介绍世界，而是让读者立刻确认：这本书兑现了书名和简介承诺。

## 一、开书前四问

开书前必须回答：

- 读者被什么点进来：题材、身份差、能力差、关系冲突、强结果承诺。
- 第一章看什么：主角困境、核心能力、冲突爆点或反常事件。
- 三章后想看什么：下一次打脸、升级、揭秘、反杀或关系变化。
- 十章后期待什么：第一次大兑现或重大反转。

任一问题答不出，先修书名/简介/前提，不写正文。

## 二、书名与简介承诺

书名和简介要建立同一个承诺，不允许互相错位。

合格书名常见结构：

- 身份差：废柴/前夫/小人物/社畜 + 隐藏能力。
- 能力差：开局获得、觉醒、绑定、重生、读心、空间、系统。
- 冲突差：被退婚、被背叛、被全网质疑、被逐出家门。
- 结果承诺：逆袭、封神、暴富、执掌、横推、全员后悔。

简介最低模板：

```text
主角处境 + 核心能力/资源 + 第一个冲突 + 读者想看的兑现方向 + 章末级悬念
```

## 三、第一章闸门

第一章必须有 4 个动作：

1. 让主角在具体场景里承受压力。
2. 让读者看见主角的特别之处。
3. 让主角做一个主动选择。
4. 用强钩子把读者推向第二章。

第一章禁止：

- 先讲世界观、等级、历史、组织百科。
- 只写主角醒来、回忆、发呆、照镜子。
- 金手指只响不兑现。
- 结尾停在环境描写或泛情绪总结。

## 四、黄金三章结构

| 章节 | 核心任务 | 爽点/钩子 |
| --- | --- | --- |
| 第1章 | 确认承诺，打破旧生活 | 小冲突 + S/A 级钩子 |
| 第2章 | 主角主动试探能力或规则 | 小爽点或明确期待 |
| 第3章 | 兑现一个期待，扩大问题 | 中钩子，开启前10章主线 |

检查规则：

- 三章内必须出现核心能力/核心资源的有效使用。
- 三章内必须让一个阻力升级。
- 三章内必须完成一次“期待 -> 延迟 -> 兑现”。
- 三章结尾必须让读者知道下一段要看什么。

## 五、前10章规划

前 10 章使用“三段推进”：

```text
1-3章：确认类型承诺
4-6章：制造连续追读
7-10章：第一次大兑现
```

## 六、开书审稿结论

审稿时只给三类结论：

- 可开：承诺清晰，第一章能兑现，前三章有追读梯度。
- 需修：核心卖点成立，但书名/简介/第一章错位。
- 暂停：只有设定，没有主角欲望、核心冲突或类型快感。

禁止只说“文笔不错但节奏慢”。必须指出慢在哪里：承诺未兑现、冲突未出现、能力未使用、钩子弱，或主角没有主动动作。
