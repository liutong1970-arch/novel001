# 长线规划与卷间衔接（L5）

长篇不能只靠单章爽点续命。v6 的长线规划用于控制主线、升级、反派梯队、伏笔回收和卷间换挡。

## 一、全书骨架

```markdown
书名：
核心类型承诺：
主角终局欲望：
主角终局代价：
核心对手/终局阻力：
核心秘密：
预计字数：
```

如果终局欲望无法说清，先不要扩展势力和地图。

## 二、节点规划

| 字数节点 | 功能 | 必须完成 |
| --- | --- | --- |
| 2万字 | 签约准备 | 前提清晰，类型快感已兑现 |
| 8-10万字 | 验证/首秀准备 | 第一轮大爽点完成，下一轮期待明确 |
| 30万字 | 长线换挡 | 新地图/新势力/新规则至少一项升级 |
| 50万字 | 防疲劳 | 回收中线伏笔，揭示更大秘密 |
| 80万字以上 | 终局牵引 | 主线向终局对手或终极真相收束 |

## 三、卷结构

每卷都按小型故事设计：

```markdown
卷名：
卷核心欲望：
卷核心阻力：
卷核心爽点：
卷中点反转：
卷低谷：
卷高潮兑现：
卷末钩子：
遗留伏笔：
下一卷承诺：
```

卷间衔接规则：

- 旧卷必须兑现一个大期待。
- 新卷必须提出更大的问题。
- 卷末不能只收束，不开启。
- 换地图必须换冲突结构，不能只换地名。

## 四、反派与阻力梯队

阻力必须升级：

```text
小阻力 -> 组织阻力 -> 规则阻力 -> 价值阻力 -> 终局阻力
```

每一层阻力都要有：

- 能压住主角的资源。
- 攻击主角弱点的方法。
- 被击败后的后果。
- 引出下一层阻力的线索。

## 五、防失控规则

出现以下情况，暂停写正文，回到长线规划：

- 主角连续 5 章目标不清。
- 连续 3 章只换地点不换冲突。
- 新增势力无法影响主角选择。
- 升级只加战力，不改变剧情解法。
- 伏笔超过预计回收章 10 章仍未推进。
