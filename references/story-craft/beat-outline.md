# 节拍与大纲

用于 `/wenxin-v6 生成15节拍`、`生成卷纲`、`生成章纲`、`大纲`。

## Save the Cat 15节拍

| 节拍 | 功能 | 长篇小说用法 |
| --- | --- | --- |
| 开场形象 | 展示主角旧世界 | 第1章或序章给出缺陷状态 |
| 主题呈现 | 暗示主角需要学什么 | 不要说教，用别人一句刺耳的话或小事件 |
| 铺垫 | 展示生活、问题、人物关系 | 埋下后续回收的伏笔 |
| 催化剂 | 旧世界被打破 | 主角不能当没发生 |
| 争论 | 主角抗拒或误判 | 展示恐惧、错误信念 |
| 进入第二幕 | 主角主动跨门槛 | 做出不可逆选择 |
| B故事 | 主题关系线 | 逼出主角内在需求 |
| 游戏时光 | 类型快感兑现 | 读者买这类故事想看的东西 |
| 中点 | 虚假胜利/虚假失败 | 风险升级，故事换挡 |
| 坏人逼近 | 外压内压夹击 | 主角弱点开始反噬 |
| 一无所有 | 失去关键支撑 | 不是小挫折，是价值跌到底 |
| 灵魂黑夜 | 主角面对真相 | 旧策略不能再用 |
| 进入第三幕 | 新认知转为新行动 | 主角主动选择代价 |
| 结局 | 解决A/B故事 | 行动证明主题 |
| 结尾形象 | 新世界 | 与开场形成对照 |

## 长篇网文分卷法

每卷都要有小型15节拍，不要只靠全书一个大结构。

```text
卷名：
卷核心欲望：
卷核心价值：
卷对手：
卷开场低点：
卷中点反转：
卷一无所有：
卷高潮选择：
卷结尾钩子：
```

## 章纲模板

```markdown
# 第X章

本章功能：
本章读者期待：
本章价值转变：
出场人物：
伏笔：
情绪值起点：
情绪值终点：

## 细纲（2-3 个深场景）
1. 场景一：目标 / 阻力 / 转变 / 钩子
2. 场景二：目标 / 阻力 / 转变 / 钩子
3. 场景三：目标 / 阻力 / 转变 / 钩子
```

## 提纲检查

- 每章是否至少推进一个：欲望线、关系线、秘密线、代价线。
- 连续三章不能只有铺垫。
- 每卷中点必须改变读者对局势的理解。
- 每卷结尾必须带来新问题，而不是只收束旧问题。
- 幕与幕之间必须不可逆，不能只是换地图。
