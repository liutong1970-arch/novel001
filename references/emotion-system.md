> 本文件示例统一使用 `[主角]`、`[配角A]`、`[伙伴]` 等占位符，落地项目时替换为本项目角色。

> 引用关系：本文件为 L4/L6 的辅助追踪模块，情绪值规则不得与 L0 五维模型和 L1 创作律冲突。

---

## 一、系统概述

情绪值系统用于量化角色的情绪状态，增强读者的代入感和爽感。

### 核心理念

- **情绪可视化**：将抽象的情绪转化为具体的数值
- **情绪驱动剧情**：情绪值变化推动剧情发展
- **情绪影响能力**：情绪值影响角色的战斗能力
- **情绪制造爽点**：通过情绪值变化制造爽点

## 三、情绪值变化规则

### 1. 情绪值触发条件

**愤怒值提升**：
- 被嘲笑（+10-20）
- 被欺负（+20-30）
- 家人被伤害（+30-50）
- 家族被陷害（+50-80）

**愤怒值降低**：
- 冷静下来（-10-20/小时）
- 使用功法（-20-30）
- 龙宠安抚（-10-20）

**悲伤值提升**：
- 家人受伤（+20-30）
- 家人牺牲（+50-80）
- 家族覆灭（+80-100）

**悲伤值降低**：
- 时间流逝（-5-10/天）
- 获得安慰（-10-20）
- 完成复仇（-30-50）

**恐惧值提升**：
- 面对强敌（+10-20）
- 生死危机（+30-50）
- 龙王威压（+20-40）

**恐惧值降低**：
- 获得力量（-20-30）
- 击败敌人（-30-50）
- 龙宠保护（-10-20）

**喜悦值提升**：
- 获得认可（+10-20）
- 实力提升（+20-30）
- 家人团聚（+30-50）
- 复仇成功（+50-80）

**喜悦值降低**：
- 遭遇挫折（-10-20）
- 失去重要的人（-50-80）

**自信值提升**：
- 获得力量（+20-30）
- 击败敌人（+30-50）
- 获得认可（+10-20）

**自信值降低**：
- 遭遇失败（-20-30）
- 被嘲笑（-10-20）
- 面对强敌（-10-20）

**杀意值提升**：
- 家人被伤害（+30-50）
- 家族被陷害（+50-80）
- 面对仇人（+40-60）

**杀意值降低**：
- 复仇完成（-50-80）
- 获得救赎（-30-50）

## 五、情绪值在写作中的应用

> ⚠️ **正文纯净规则**：情绪值标注只用于章节细纲和追踪表（`追踪/情绪追踪.md`），**正文文件禁止出现情绪值标注、Markdown表格、"注："等元信息**。正文中不允许出现情绪值表格。写正文时通过人物动作、神态、心理描写来体现情绪，不直接标注数值。

> ⚠️ **禁用模式**：情绪值示例不得使用 L1 列出的一级/二级/三级禁词。所有示例必须符合 L1 创作律。

### 1. 情绪值标注格式

在章节细纲中标注情绪值变化：

```
【场景1：被嘲笑】
- 主角被对手嘲笑
- 情绪变化：愤怒+20，自信-10
- 当前情绪值：愤怒35，自信25

【场景2：反击】
- 主角反击成功
- 情绪变化：喜悦+30，自信+20
- 当前情绪值：喜悦35，自信45

【场景3：家族危机】
- 家族面临威胁
- 情绪变化：愤怒+50，悲伤+40，杀意+30
- 当前情绪值：愤怒85，悲伤40，杀意30
```

### 2. 情绪值描写示例

示例只展示“细纲情绪值如何转成正文动作”，正文不得出现数值。

**愤怒30（平静）**：
```text
[主角]把杯沿擦干净，推回桌子中央。
[配角A]还在笑。
他没有接话，只把椅子往后挪了半寸。
```

**愤怒50（恼怒）**：
```text
[主角]的手压在桌边。
木纹硌着掌心，他没有松开。
"再说一遍。"
```

**愤怒70（愤怒）**：
```text
[主角]跨过地上的碎瓷片。
鞋底碾出轻响。
他停在[配角A]面前，挡住了对方看向[伙伴]的视线。
```

**愤怒90（暴怒）**：
```text
[能力]从指缝里漏出来，贴着地面铺开。
[主角]没有吼。
他抬手扣住[配角A]的腕骨，把那只手从[伙伴]身边按开。
"这次轮到你退。"
```

**恐惧30（警惕）**：
```text
[主角]站在门内，先看窗，再看地上的影子。
风从门缝挤进来，带着铁锈味。
```

**恐惧50（紧张）**：
```text
[主角]退到柜边，指尖摸到冰凉的铜扣。
他没有回头。
那道脚步声停在门外，离他只隔一层木板。
```

**恐惧70（恐惧）**：
```text
[主角]的膝盖撞到床沿。
疼意往上窜，他借着那一下稳住身体。
[伙伴]在被褥里叫了一声，比刚才更轻。
```

**恐惧90（恐慌）**：
```text
[主角]拽起[伙伴]，撞开侧门。
门轴刮过肩膀，他顾不上疼。
身后的脚步声变快了。
```

## 七、情绪值写作技巧

### 1. 情绪值递进

**错误写法**：
```
[主角]很愤怒。他冲上去打倒了[配角A]。
```

**正确写法（情绪递进，正文不标数值）**：
```
[主角]看了[配角A]一眼。
他攥紧拳头。
指节在袖口里顶出一排白痕。
他向前一步，把[配角A]逼回门边。
```

### 2. 情绪值对比

**细纲记录（不进入正文）**：
```
之前：[主角]被嘲笑，愤怒35，自信25。
之后：[主角]击败[配角A]，愤怒10，自信80。
```

**正文转写**：
```
[配角A]的笑声停了。
[主角]把手收回来，袖口垂下去，遮住发红的指节。
```

### 3. 情绪值爆发

**情绪爆发制造高潮**：
```
[主角]抬起眼。
屋里的笑声像被一只手掐断。
[能力]贴着地面铺开，逼得[配角A]后退半步。
```

### 4. 情绪值控制

**情绪控制展现成长**：
```
[主角]闭上眼睛，让呼吸慢下来。
掌心里的[能力]一点点收回经脉。
他睁开眼睛，眼神冰冷。
"我不杀你。"
```
