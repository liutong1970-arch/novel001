# 方想风格指纹（L2，通过 L1 检验）

## 声明

本风格为多风格库中的可选项之一。写作时单选一种风格，不混搭。选用本风格时，全文按本指纹生成。

本文件所有范例均按本项目创作律（L1）规范化，非原书逐字。

## 五维指纹

| 维度 | 偏向 | 说明 |
|------|------|------|
| 突发性 | 极高 | 短句为骨架（3-8 字无主语句高频），技术设定段穿插长句，形成强方差 |
| 困惑度 | 高 | 设定专业词（光甲/感知/制卡/能量断层）硬性并置，词链不可预测 |
| 信息熵 | 高 | 无主语短句 + 物理量术语，句首/句式多变 |
| 追读力 | 中 | 节奏偏慢，适合科幻/机甲类，爽点密度中等 |
| 平台留存力 | 中低 | 适合已有耐心读者和科幻/机甲受众，开书需额外补强书名承诺、前三章兑现和章末钩子 |

## 文风内核

### 1. 空间性叙事 / 线性断裂

拒绝时间轴平滑过渡。时空交叉、章节并置，制造叙事盲点与偶然性游记色彩。前后事件可以缺乏必然因果，留下大量叙事盲点后文再回收。

### 2. 动作帧切片 / 机械美学

战斗和操作描写拆解为一帧帧物理动作：身体部位的空间位移、重力转移、物理反馈。无形容词，无心理概括。

正例：

```text
左腿后撤半步，腰下沉，右肘内收，剑尖上扬，目光与剑尖平齐。
```

反例（含概括和形容词）：

```text
他的动作极其敏捷，身形如鬼魅般闪过对手的攻击。
```

### 3. 不平衡节奏

日常段落节奏慢，允许长段铺陈技术设定、机械参数、地理环境。冲突爆发时节奏陡然加速，句型急剧缩短，信息密度骤增，制造极速流的阅读体验。

## 仿写参数

- **短句为骨架**：3-8 字无主语短句占比目标 ≥35%。
- **长句调剂**：技术/机械参数段落允许 20-40 字长句，服务突发性（引用 L0：长短交错，非"禁止长句"）。
- **去逻辑词**：禁用"然而/因此/由此可见/不仅如此"等（引用 L1 去高概率词链律）。
- **对白**：遵循 L1 对白律，默认无标签。角色区分靠词汇、句长、语气。
- **词汇**：引入项目专有设定词（机甲型号、功法名、能量单位），提升信息熵（引用 L1 术语锚定律）。
- **情感**：极度冷淡，聚焦于角度、重力、能量流转等物理量描述。角色情绪通过身体反应和错误判断露出，不靠形容词。

## 范例（规范化后）

### 战斗切片

```text
他右脚蹬地，膝关节微屈，重心压到左胯。
剑身平移，刃口对准对方肩胛骨下沿。
对方侧身，左臂格挡，金属碰撞的声音在走廊里弹了两下。
他没有停。剑尖顺势下滑，切过护臂边缘，带出一道浅浅的血痕。
对方后退三步，右脚跟撞到墙根。
```

### 技术设定铺陈（长段）

```text
一星卡的制作流程不算复杂：先在空白卡基上用蚀刻笔画出基础回路框架，再依次注入三种不同频率的能量脉冲，每种脉冲的间隔必须控制在零点三到零点五秒之间，过快会导致回路过载烧毁卡基，过慢则能量无法在节点处形成稳定共振。陈暮每天重复这个流程四十遍以上，手指关节磨出了茧，蚀刻笔换了三根。
```

### 日常段落

```text
陈暮坐在工坊角落，把今天的废卡一张张叠好。
七十三张。昨天是六十八张。
他把废卡摞推到桌角，拿起蚀刻笔，换了一张新卡基。
```

## 与防 AI 痕迹的天然合流

方想文风在五维模型上有明显优势与短板：

- 短句拉突发性 → 句长方差大。
- 设定词拉熵 → 词汇丰富度高。
- 去过渡词拉困惑度 → 词链不可预测。
- 机械美学和动作切片拉追读力 → 信息增量密度高。
- 平台留存力偏弱 → 开书必须额外强化书名承诺、黄金三章和章末钩子。

选方想风格，"写得好"与"测不出"容易统一，但若用于番茄平台，需要单独补强留存入口。
